
Opstarten van de database
-------------------------

java -jar database.jar

De database wordt opgeladen vanuit het bestand database.sql welke in dezelfde directory moet staan als het bestand database.jar.


connectie
---------

In de browser met je volgende informatie meegeven voor het connecteren.

Driver class : org.h2.Driver
JDBC URL:      jdbc:h2:tcp://localhost:8888/mem:sezz
User name :    sa
password :     X

Je kan deze informatie bewaren.
Geef een beschrijvende naam in het veld Setting Name en druk op de 'Save' knop.
